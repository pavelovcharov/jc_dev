Файловый менеджер для Windows и Linux


JC - это файловый менеджер, с двумя панелями, кроссплатформенный аналог NC и FAR.

Основные возможности:
  • работа в любой операционной системе при наличии виртуальной машины Java;
  • удобная навигация по файловой системе;
  • просмотр папок во вкладках;
  • запуск файлов, копирование, удаление, перемещение, поиск файлов и папок;
  • поиск по маске и по содержимому файлов;
  • просмотр и редактирование файлов, внутренний редактор;
  • поддержка работы с сетью (с использованием библиотеки jcifs-1.2.10);
  • внутренний распаковщик для архивов zip/jar/tar/tar.gz/tgz;
  • внутренний zip-упаковщик;
  • внутренняя программа просмотра  изображений(jpg, gif, png, bmp);
  • поддержка пользовательских "горячих" клавиш;
  • поддержка многоязычного интерфейса.
Для работы приложения необходима виртуальная машина Java версии не ниже 1.5 (рекомендуется Java 1.6). Последние версии доступны по адресу http://www.java.com.
